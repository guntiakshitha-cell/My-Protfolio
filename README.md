# My Portfolio Website

## Overview

This is my personal portfolio website that showcases my projects, skills, and contact information. The website is designed to present my work and help recruiters or collaborators understand my technical abilities.

## Features

* Home section introducing myself
* Projects section displaying my work
* Skills section highlighting my technical skills
* Contact form for sending messages
* Links to my GitHub and LinkedIn profiles and email

## Technologies Used

* HTML
* CSS
* JavaScript

## Project Structure

portfolio
│
├── index.html
├── style.css
├── script.js
└── assets/

## How to Run the Project

1. Download or clone the repository.
2. Open the project folder.
3. Run `index.html` in any web browser.
